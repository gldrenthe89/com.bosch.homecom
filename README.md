# Bosch HomeCom Easy Community

Homey app for Bosch K30/K40 boilers and heat pumps via the Bosch HomeCom Easy cloud.

---

**Important:**
- This app **only supports K30/K40 devices**. We do not have access to other Bosch devices.
- This is an **unofficial community app**. Bosch does not support this integration.
- The integration may stop working at any time due to changes in the Bosch API.
